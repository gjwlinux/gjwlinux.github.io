em 倾斜 单标签

td 表格中tr中行，双标签
caption 表格标题，双标签
a href 超链接，双标签
a target=”blank“ 配合href 点开后跳转新建标签页面
img src 添加图片
&#nbsp; 空格

python列表常用方法

发表于 2019-05-21 | 分类于 python

  • 查看列表有哪些方法

  • append列表中追加内容

  • count统计列表中元素出现的次数

  • extend追加内容，但是不会出现列表嵌套结构，append会嵌套

  • index 返回索引位置

  • insert 插入内容，指定索引位置

  • pop 删除列表内容，按照索引位置，返回删除信息

  • remove 移除列表内容，不反回移除信息

  • reverse 反转列表

  • sort 排序

python字符常用方法

发表于 2019-05-21 | 分类于 python

字符串方法

  • 查看字符串有哪些方法

  • count 统计字符串中出现的个数

  • startswith 查看字符串是不是以什么开头,返回布尔值

  • endswith 查看字符串是不是以什么结尾，返回布尔值

  • find 检查字符串中是否有字符，指定开始与结尾，不存在返回-1

  • format 格式化输出

  • index 查找字符串索引位置

  • isdigit 是不是数字组成，返回布尔值

  • islower 是不是小写，返回布尔值

  • isupper 是不是大写，返回布尔值

  • join 连接字符串

  • ljust 字符串左对齐,默认使用空格填充至指定长度的新字符串

  • lower 将字符串转换为小写

  • lstrip 去掉字符串左边空格，或指定字符

  • replace 替换字符串

  • rjust 字符串右对齐,默认使用空格填充至指定长度的新字符串

  • split 字符串切分返回列表

  • strip 去除指定字符

  • upper 将字符串变为大写
